# **App Name**: ShopLocal

## Core Features:

- Product Browsing: Browse and filter products by category.
- Product Search: Search for products by name or category.
- Shopping Cart: Add, remove, and update product quantities in the cart.
- Secure Checkout: Simulate cash on delivery and fake card payments to finalize orders.
- Order History: View a list of previous orders with details.
- Admin Product Management: Add, edit, and delete products, including image uploads, through the admin panel.
- Admin Order Management: View and update order statuses through the admin panel.

## Style Guidelines:

- Primary color: Soft teal (#A0D6B4) for a calm, trustworthy shopping experience.
- Background color: Light greenish-gray (#F0F4F2).
- Accent color: Pale yellow (#D6CA98) for call-to-action buttons.
- Headline font: 'Poppins', a sans-serif font to provide a modern feel.
- Body font: 'PT Sans', a sans-serif font for clear and readable content.
- Simple and modern icons for navigation and product categories.
- Clean and responsive grid layout for products and pages. Sidebar navigation for admin panel.
- Subtle animations for transitions and loading states to enhance user experience.
